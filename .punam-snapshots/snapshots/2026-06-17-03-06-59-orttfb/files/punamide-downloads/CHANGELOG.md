# Changelog

All notable public PunamIDE alpha releases will be documented here.

## v0.1.0-alpha

### Added

- AI Chat.
- Native Snapshots.
- Technical Debt Analysis.
- Git Integration.
- Terminal Integration.
- Error Reporting.
- Local Log Export.
- Diagnostics generation.
- Alpha support and feedback workflows.

### Known Issues

- Windows alpha only.
- Some workflows may be incomplete or unstable.
- AI provider setup may require manual configuration.
- Terminal behavior may vary across systems.
- Snapshot restore should be tested carefully before relying on it for critical work.
- Error reporting and diagnostics are still being refined.

### Future Plans

- Improved installer and updater workflow.
- Better first-run onboarding.
- More complete snapshot history and restore controls.
- Expanded technical debt intelligence.
- Stronger Git diff and review workflows.
- More AI provider integrations.
- Better documentation and community support resources.
- Additional platform support after Windows alpha stabilizes.
