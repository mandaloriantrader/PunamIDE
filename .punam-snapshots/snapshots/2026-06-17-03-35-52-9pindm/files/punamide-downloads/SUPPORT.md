- Download questions.
- General support.

## Bug Reports

Use GitHub Issues.

When reporting bugs, include:

- PunamIDE version.
- Windows version.
- Steps to reproduce.
- Screenshots.
- Logs or diagnostics if available.

## Feature Requests

Use GitHub Discussions.

Helpful feature requests explain:

- The problem.
- The desired workflow.
- Why it matters.
- Any alternatives you tried.

## Alpha Notes

PunamIDE is early alpha. Some issues are expected.

Before using PunamIDE on important projects:

- Use Git.
- Keep backups.
- Use PunamIDE native snapshots before risky AI edits.
