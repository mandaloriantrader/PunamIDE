# PunamIDE

![PunamIDE logo placeholder](assets/logo.png)

**Build fast. Break less. Recover instantly.**

PunamIDE is an AI-powered desktop IDE built with Rust and Tauri for developers who experiment aggressively.

Most IDEs added AI.  
**PunamIDE was built around surviving AI mistakes.**

Website: [https://punamide.com](https://punamide.com)  
Discord: [Join the Alpha Discord](https://discord.gg/PFp9KWY3eY)

> PunamIDE is currently in early alpha. Windows builds are released first. Source code is private during the alpha period; this repository is used for public downloads, release notes, support, and community feedback.

## Why PunamIDE Exists

AI can speed up development, but it can also break working code, introduce hidden regressions, and leave projects harder to reason about.

PunamIDE is designed for the messy reality of AI-assisted development:

- Try bold changes.
- Review what changed.
- Understand technical debt.
- Recover quickly when something breaks.
- Keep project workflows inside one focused desktop IDE.

## Key Features

### Native Snapshots

Take project snapshots before risky AI edits, refactors, experiments, or dependency changes.

If AI breaks your code, PunamIDE helps you recover without panic:

- Create native workspace checkpoints.
- Export snapshots as zip backups.
- Restore a previous state anytime.
- Keep experimentation safe during alpha testing.

### Technical Debt Intelligence

PunamIDE includes technical debt analysis tools to help you understand where a project is fragile, complex, or likely to need refactoring.

Alpha features include:

- Debt scoring.
- Module breakdowns.
- Refactor queues.
- Dependency and complexity signals.

### AI Chat

PunamIDE includes project-aware AI chat designed for code understanding, debugging, and workflow support.

Use it to:

- Ask questions about your project.
- Understand file structure.
- Debug runtime and build errors.
- Plan changes before editing.

### Git Integration

PunamIDE includes built-in Git workflows so you can inspect changes without leaving the IDE.

Alpha support includes:

- Project change awareness.
- Diff-oriented review workflows.
- Git context for development tasks.

### Terminal Integration

Run commands inside PunamIDE and keep terminal output close to your project context.

Alpha support includes:

- Integrated terminal workflows.
- Command output visibility.
- Build and runtime debugging support.

### Error Reporting and Diagnostics

PunamIDE includes early alpha diagnostics designed to make bug reports useful.

Features include:

- Error reporting.
- Local log export.
- Diagnostics generation.
- Crash-friendly error screens.

## Alpha Warning

PunamIDE is early alpha software.

Expect:

- Bugs.
- Incomplete workflows.
- UI changes.
- Occasional crashes.
- Features changing between releases.

Before using PunamIDE on important projects, create backups or use Git. Native snapshots help, but alpha software should still be treated carefully.
