# Code of Conduct

## Our Pledge

We want the PunamIDE community to be welcoming, constructive, and professional.

Everyone participating in PunamIDE spaces is expected to treat others with respect, regardless of experience level, background, identity, or viewpoint.

## Expected Behavior

Examples of expected behavior:

- Be respectful and constructive.
- Assume good intent.
- Give actionable feedback.
- Stay focused on the topic.
- Help alpha testers feel welcome.
- Report bugs clearly and calmly.

## Unacceptable Behavior

Examples of unacceptable behavior:

- Harassment or intimidation.
- Personal attacks.
- Hate speech.
- Sexualized language or imagery.
- Sharing private information without consent.
- Repeated disruptive behavior.
- Abusive comments toward maintainers, users, or testers.

## Enforcement

Maintainers may remove content, close issues, restrict Discord participation, or block users who violate this code of conduct.

## Reporting

Report concerns through Discord:

[https://discord.gg/PFp9KWY3eY](https://discord.gg/PFp9KWY3eY)

Reports will be reviewed as fairly and promptly as possible.

## Scope

This code of conduct applies to:

- GitHub Issues.
- GitHub Discussions.
- Discord.
- Release comments.
- Any official PunamIDE community space.
